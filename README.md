# About

**LeNeMon** is the official Fabric mod developed for the **LeNeMon Minecraft server**.

The mod enhances the player experience with custom systems, server integrations and exclusive gameplay mechanics designed specifically for the server ecosystem.

It integrates custom interfaces, server utilities and gameplay enhancements that are required to access all server features.

Official download page
[https://www.curseforge.com/minecraft/mc-mods/lenemon-fabric](https://www.curseforge.com/minecraft/mc-mods/lenemon-fabric)

---

# Features

The mod currently includes multiple gameplay systems and server integrations.

Main features include:

* Custom main menu
* Direct connection to the LeNeMon server
* Integrated Discord button
* Exclusive armor sets with special effects
* Custom GUI systems
* Server gameplay mechanics
* Various utilities designed for the LeNeMon ecosystem

More features may be added in future updates.

---

# Requirements

To run the mod correctly you need the following environment.

* Minecraft **1.21.1**
* **Fabric Loader**
* **Fabric API**
* **GeckoLib**
* **Java 21 or newer**

Fabric documentation
[https://fabricmc.net/wiki](https://fabricmc.net/wiki)

---

# Installation

Client installation

1. Install Fabric Loader
2. Install Fabric API
3. Download the latest LeNeMon release
4. Place the `.jar` file in your `mods` folder
5. Launch Minecraft using the Fabric profile

Server installation

The mod must also be installed on the server to enable all server-side systems.
